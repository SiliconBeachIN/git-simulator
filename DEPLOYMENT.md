# Deployment Checklist — gitsimulator.xyz

This checklist helps with deploying `GitSimulator` to a static host and verifying SEO and TLS.

1. DNS
   - Add an A record (or ALIAS) pointing your apex domain `gitsimulator.xyz` to your host's IP (or provider-specific target).
   - Or add a `CNAME` for `www` to your host's provided domain and redirect apex to `www` as desired.

2. Hosting
   - Recommended hosts: Netlify, Vercel, GitHub Pages, or any static host that serves `dist/`.
   - In your host settings, set the custom domain to `gitsimulator.xyz` and ensure the host has a generated TLS certificate.

3. TLS / HTTPS
   - Enable automatic TLS (Let's Encrypt) in your hosting provider dashboard. Verify `https://gitsimulator.xyz` loads with a valid certificate.
   - If using Cloudflare or another CDN, ensure proxy/caching preserves headers and serves the TLS cert correctly.

4. CNAME and redirects
   - `public/CNAME` (already present) contains `gitsimulator.xyz` for GitHub Pages/Netlify CNAME support.
   - Ensure any non-www → www redirects (or vice versa) are configured at the host or DNS provider.

5. Build & assets
   - Locally: run `npm install` then `npm run build`. The production files appear in `dist/`.
   - Upload or connect your repo so the host runs `npm run build` during deployment (or upload `dist/`).

6. Sitemap & robots
   - `public/sitemap.xml` is generated by `scripts/generate-sitemap.mjs` and references `https://gitsimulator.xyz`.
   - Ensure `public/robots.txt` points to the sitemap URL.

7. Social images & Open Graph
   - Verify `dist/*/index.html` pages have `og:image` and `twitter:image` URLs pointing to `https://gitsimulator.xyz/social/*.png` and that those images are published.

8. Analytics & Search Console
   - Add the recommended Google Analytics / GTM IDs to `index.html` (already present) if desired.
   - Submit your sitemap to Google Search Console and Bing Webmaster Tools.

9. Post-deploy checks
   - Visit: https://gitsimulator.xyz/  — verify canonical `<link>` and OG tags contain HTTPS domain.
   - Check: https://gitsimulator.xyz/sitemap.xml  — verifies sitemap is reachable.
   - Run a Lighthouse audit and verify no mixed-content warnings.

10. Optional
   - Enable Brotli/Gzip at CDN or host for smaller transfers.
   - Configure cache headers for static assets.
